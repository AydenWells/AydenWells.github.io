CSCE 242: Website Applications

Ayden Wells Mathews

Web technologies to support client-server computing. Implementation of client-server applications. Firstly, students will utilize HTML and CSS to design the front-end of a website; matching a given specification. Secondly, they will use JavaScript to add functionality to their website. Lastly, they will learn to utilize node and server side JavaScript to store and retrieve data from a server.
